# Afazeres Projeto · v1.0

Gerenciador de tarefas diárias feito para uso pessoal. Surgiu da necessidade de parar de depender de papel e ter algo simples e funcional sempre à mão.

## Funcionalidades

- Persistência via **LocalStorage** — as tarefas continuam lá depois de fechar o navegador
- Adição rápida pelo botão ou tecla `Enter`
- Check com efeito visual de conclusão (strikethrough)
- Duplo clique para excluir um item

## Tecnologias

`HTML5` · `CSS3` · `JavaScript Vanilla`

## Como usar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/afazeres-projeto

# Abra direto no navegador
open index.html
```

## Contexto de desenvolvimento

Projeto iniciado por mim — estrutura visual e escopo definidos desde o começo. Para agilizar a lógica de persistência e manipulação do DOM, usei o Gemini como suporte. A base pronta, ajustei o comportamento e o design até ficar do meu jeito.

> *"Odeio usar agentes de IA pra programar, mas pela necessidade de agilidade, foi a solução ideal pra não deixar a ideia só no papel."*
